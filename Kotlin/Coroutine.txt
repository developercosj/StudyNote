- suspendable computation
    - async/await
    - 퓨처 (future)
    - 프라미스 (promise)
    - 액터 (actor)

- 코루틴의 주요 기능
    - 스레드에 대한 더 가벼운(lightweight) 대안
    - 유연한 스레드 디스패치 매커니즘
    - 일시 중단 가능한 시퀀스와 이터레이터(iterator)
    - 채널(channel) 을 사용한 메모리 공유
    - 액터를 통해 메시지 전송 방식으로 상태 공유

- 변수
    - 지역변수 : 함수 안에서 지정하는 변수
    - 프로퍼티 : 값을 읽거나 쓸 때 계산을 수행 가능
    - 타입추론(type inference) : 컴파일러가 코드의 문맥에서 타입을 도출해주는 언어 기능
    - 불변 변수(자바의 final 과 비슷)와 가변 변수
        #가능하면 불변 변수를 많이 사용해야 한다.

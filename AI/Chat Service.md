# 업무자동화 
AI 산업에서의 업무자동화(Workplace Automation) 는 단순히 반복적인 클릭을 대신해주는 수준을 넘어
판단과 생성의 영역으로 빠르게 확장되고 있다. 

1. 지능형 자동화(LLM 기반 에이전트)
Ai Agents : 사용자의 목표를 이해하고 실행 계획세워 도구 사용 
예시 (일반 사용자용)
    1-1. Anthropic 의 Cowork 
    1-2. 뤼튼(Wrtn) & 라이너(Liner) : 뤼튼의 슈퍼앱은 사용자의 일정을 기억하고 맞춤형 제안을 하거나 리서치를 수행하는 에이전틱 기능 강화 
    1-3. NotebookLM: 구글, 문서를 학습해 스스로 통찰 도출 및 오디오 브리핑까지 만들어주는 연구 전용 에이전트 역할 
예시 (업무 및 생산성용)
    1-4. 개인 비서,Microsoft Copilot Studio,"윈도우/오피스 환경에서 메일 작성, 일정 예약, 문서 요약 등을 알아서 처리합니다."
    1-5. 워크플로우,Gumloop / Relay.app,'Zapier의 AI 버전'이라 불립니다. 여러 앱을 연결해 복잡한 업무 흐름을 스스로 실행하게 만듭니다.
    1-6. 리서치,Perplexity / OpenAI Operator,"단순히 검색 결과를 보여주는 게 아니라, 웹 서핑을 통해 정보를 취합하고 보고서를 작성합니다."
    1-7. 협업/기획,Atlassian Rovo,Jira나 Confluence 내에서 프로젝트 현황을 분석하고 이슈를 대신 처리해줍니다.
예시 (개발/기술용)
    1-8. Devin AI: "세계 최초의 AI 소프트웨어 엔지니어"로 불립니다. 요구사항만 주면 코딩, 디버깅, 배포까지 전 과정을 스스로 수행합니다.
    1-9. Claude Code: 개발자들을 위한 터미널 기반 에이전트로, 복잡한 코드 수정 작업을 자율적으로 처리합니다.
    1-10. CrewAI / Autogen: 여러 명의 AI 에이전트(예: 마케터 에이전트, 개발자 에이전트, 검수 에이전트)가 팀을 이뤄 협업하도록 설계할 수 있는 프레임워크입니다.


n8n : 에이전트 오케스트레이션(Agent Orchestration) 플랫품 -> 시각적(GUI)으로 에이전트의 사고 과정과 행동을 설계 
다른 종류 : 
- Dify : 현재 n8n과 함께 가장 핫한 오픈소스 플랫폼 
- Flowise : 개발자들에게 유명한 LangChain 을 드래그 앤 드롭 방식으로 바꾼 툴 / 복잡한 AI 논리 구조를 시각적으로 설계할 때 유리  
- LangFlow / Stack AI
- Stack AL : 기업용으로 설계된 보안 중심 빌더 
Dify : n8n 과 비슷하게 노드를 연결하는 방식, AI 에이전트를 만드는데 필요한 기능들이 더 잘 갖춰짐

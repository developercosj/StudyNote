- 전화번호 숫자 입력 시 - 기호 제거하고 숫자만 남기기
    - .replaceAll(“[^0-9]”, “”);
- toCharArray() 메서드
    - 문자열을 한글자씩 쪼개서 이를 char 타입의 배열에 집어 넣어준다.
    - String 문자열을 char 형 배열로 바꿔줌
    - for(char c : number.toCharArray()) {} 형태로 작성 가능함
- Character.isDigit(검사하고 싶은 값)
    - 명시된 char 값이 숫자인지 검사해서 true/false 값 반환
- contains(”검사하고싶은문자열”)
    - 문자열포함 여부 검사
    - 대소문자 구별함
    - 공백역시 체크함
